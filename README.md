### **Method 1: Live Website**

Simply visit: [https://tanzeel804.github.io/E-com-Demo/](https://tanzeel804.github.io/E-com-Demo/)

### **Method 2: Local Installation**

1. **Clone the repository:**

   ```bash
   git clone https://github.com/Tanzeel804/E-com-Demo.git
   ```

2. **Navigate to project folder:**

   ```bash
   cd E-com-Demo
   ```

3. **Open the website:**

   - Double-click `index.html` OR
   - Use a local server:

     ```bash
     # Using Python
     python -m http.server 8000

     # Using Node.js (if you have http-server)
     npx http-server
     ```

4. **Visit in browser:**
   ```
   http://localhost:8000
   ```
