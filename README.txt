Tanzeel Store - Demo project
Files:
- index.html : main site
- profile.html : profile page
- terms.html : terms & conditions
- css/styles.css : styles
- js/app.js : main JS (fetches Fake Store API)
- assets/* : placeholder images

How to use:
1. Download and unzip.
2. Open index.html in browser or host on GitHub Pages.
3. To replace images, put real files into assets/ and update paths.
